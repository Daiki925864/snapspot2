# アプリケーション名
Snap Spot

# アプリケーション概要
画像と撮影場所を共有するためのアプリ。何気ない写真を共有し、他のユーザーとの交流を楽しむことができる。

2. % cd snapspot2
3. % bundle install
4. % yarn install
